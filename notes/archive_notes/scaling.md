# Scaling, Progression & Prestige

## Tier Progression Structure (3.4.1)

```
TIER 1 — Normal
├─ 8 procedural dungeons (3 bosses each)
│   └─ Loot ilvl 50–100
└─ Raid Tier 1 (4–6 bosses)
    └─ Loot ilvl 90–120

TIER 2 — Heroic
├─ Same 8 dungeons, harder
│   └─ Loot ilvl 110–150
└─ Raid Tier 2
    └─ Loot ilvl 140–170

TIER 3 — Mythic
├─ Same 8 dungeons, hardest
│   └─ Loot ilvl 160–200
└─ Raid Tier 3
    └─ Loot ilvl 190–220

TIER 4 — Mythic+
├─ Infinite scaling keys
└─ Raid Tier 4
    └─ Loot ilvl 210–250

PRESTIGE RESET
└─ Clears dungeon progress, rewards permanent skill points
```

Heroic unlocks after clearing Raid Tier 1. Mythic after Raid Tier 2. Mythic+ after Tier 3.

## Failure & Retry (3.4.4)

- **Wipe = full dungeon reset.** Return to entrance, same layout and same boss roster.
- No checkpoint system.
- Between attempts: freely swap team comp, change gear, respec.
- Dungeons persist (same layout) until cleared — then a new one generates.

## Prestige System (3.5.1)

**Trigger:** Player voluntarily prestiges after clearing Raid Tier 4.

**Rewards (permanent, carry across all prestiges):**
- +10 skill points per prestige
- Expanded mechanic pool for future dungeons (new mechanic types unlock)
- Post-prestige bosses may have elemental immunities, forcing alt-character use
  - Example: "Frostlord Glacius" immune to frost → must bench the Frost Mage

**What resets:**
- Dungeon progress (back to Tier 1 Normal)
- Gear (keeps 1 best-in-slot set per character, rest discarded)

**What persists:**
- Full hero roster
- All prestige-earned skill points
- Gold and materials (25% carry-forward)

## Offline Gains (3.5.2)

Capped at 8 hours. Generates currency only — does not auto-clear dungeons.

```
Offline Gold      = (Avg dungeon gold) * (Time offline / Avg clear time) * 0.5
Offline Materials = Similar formula, reduced rate
```

Active play benefits:
- 2× gold and material rate vs offline
- Chance for bonus drops
- Ability to adjust strategy mid-fight

**Speed controls:** 1× (learning), 2× (farming), 4× (repeat clears)
